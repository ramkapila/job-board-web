# Job Board App

A full-stack job board platform where employers can post job listings and job seekers can create profiles, search for jobs, apply, and save listings.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Vite
- **Backend:** Node.js, Express
- **Database:** MongoDB (Atlas)
- **Auth:** JWT

## Features
- Employer: Register, post/manage job listings, view applicants, update application status
- Job Seeker: Register, build profile, search & filter jobs, apply with cover letter, bookmark jobs

## Getting Started

```bash
# Backend
cd backend
node server.js

# Frontend
cd frontend
npm run dev
```

> Add your MongoDB Atlas connection string and JWT secret to `backend/.env` before running.
