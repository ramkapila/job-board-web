## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/ramkapila/job-board-web.git
cd job-board-web
```

### 2. Backend Setup
```bash
cd backend
npm install
# Create .env file (see .env.example)
node server.js
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### 4. Access the app
- Frontend: http://localhost:5173
- Backend: http://localhost:5000

## 🎯 Key Features

✅ JWT Authentication with role-based access  
✅ Employer job posting with expiry dates  
✅ Job seeker profile with PDF resume upload  
✅ Search & filter jobs by title, location, type  
✅ Application tracking with status updates  
✅ Bookmark/save jobs  
✅ Employer analytics dashboard  
✅ Rate limiting on auth routes  
✅ Automated job expiry with cron  
✅ Dark mode UI with WebGL shader effects

## 📊 Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Three.js  
**Backend:** Node.js, Express, MongoDB Atlas  
**Auth:** JWT, bcryptjs  
**File Upload:** Multer  
**Security:** express-rate-limit
